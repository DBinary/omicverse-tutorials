# Enrichment

Tutorials for gene-set enrichment scoring across the `omicverse.es`
(`ov.es`) family of GPU-accelerated kernels: rank-based recovery curves
(`aucell` / `gsea` / `gsva`), linear regression (`ulm` / `mlm` / `waggr`),
over-representation (`ora`), reference statistics (`zscore`), regulon
activity (`viper`), and gradient-boosted decision trees (`mdt` / `udt`).
