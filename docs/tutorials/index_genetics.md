# Genetics

Tutorials for the `omicverse.genetics` module — a unified statistical-genetics framework
linking genotype, expression and complex traits. Covers eQTL mapping, statistical
fine-mapping (SuSiE), colocalization of GWAS and QTL signals (coloc), Mendelian
randomization, single-cell disease-relevance scoring (scDRS), SNP-heritability and
cell-type enrichment (LD score regression / LDSC-SEG), transcriptome-wide association
(TWAS / S-PrediXcan), and GWAS quality control and association testing. The statistical
engines are the standalone R-parity packages `pymatrixeqtl`, `pysusie`, `pycoloc`,
`pytwosamplemr`, `pyscdrs`, `pyldsc` and `pytwas`.
