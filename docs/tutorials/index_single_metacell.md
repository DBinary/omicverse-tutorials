# MetaCell

Metacells — small, transcriptionally homogeneous groups of single cells used
as the unit of analysis. Three layers: a recommended single-sample workflow,
a multi-sample (batch-corrected) workflow, and a backend zoo comparing all
seven `ov.single.MetaCell` partitioners.
