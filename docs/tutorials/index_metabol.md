# Metabolomics

Tutorials for the `omicverse.metabol` module — downstream analysis of peak-intensity tables
produced by XCMS / MZmine / MS-DIAL / MetaboAnalyst. Covers QC, imputation, normalization,
univariate and multivariate statistics, pathway enrichment (ORA + GSEA + mummichog), and
lipidomics-specific workflows (LIPID MAPS parsing + LION ontology).
